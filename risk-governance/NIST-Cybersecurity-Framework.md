# NIST Cybersecurity Framework Implementation

This project applies the NIST Cybersecurity Framework to a banking environment.

## Overview
The project focuses on identifying assets, assessing risks, and implementing security controls using the five NIST functions: Identify, Protect, Detect, Respond, and Recover.

## Key Areas Covered
- Asset identification and classification
- Risk assessment and threat analysis
- Access control and system protection
- Incident detection and response planning
- Business continuity and recovery strategies

## Impact
- Improved organizational security posture
- Structured approach to cybersecurity risk management
- Alignment with industry best practices

## Skills Demonstrated
- Risk management
- Security architecture design
- NIST framework application
- Governance and compliance
